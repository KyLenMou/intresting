# DreamAtlas MVP (AI 梦境地图)

一个可运行的前端初版，包含：

- 梦境地图总览（5 个区域）
- 拖拽平移 + 滚轮缩放
- 点击区域后“飞入”聚焦
- 任务面板与 AI 旁白文案
- 记忆碎片收集（localStorage 持久化）

## 快速运行

```bash
python3 -m http.server 4173
```

然后打开：`http://localhost:4173`

## 下一步建议

1. 将静态区域改为 Canvas / WebGL 地形与粒子。
2. 为每个区域接入独立场景和背景音。
3. 把旁白改为后端 AI 接口，加入分支叙事与日志。

## 云端访问（临时分享）

```bash
./scripts/cloud_share.sh 4173
```

运行后会返回一个公网临时链接，可直接分享访问（终端关闭后失效）。
